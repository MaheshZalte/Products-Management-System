## üåü Overview
**ProductHub** is a lightweight web-based CRUD (Create, Read, Update, Delete) application for managing products.  
It is built using **Spring Boot**, **Thymeleaf**, and **JPA/Hibernate**, featuring a clean and responsive UI for a smooth user experience.

---

## ‚ú® Features
‚úÖ Add new products with validation  
‚úÖ View all products in a stylish data table  
‚úÖ Edit and update product details easily  
‚úÖ Delete products instantly  
‚úÖ Beautiful UI using custom CSS  
‚úÖ Backend powered by Spring Boot & Hibernate ORM
